To run this dashboard, please install the following Python dependencies:

- streamlit
- pandas
- numpy
- pydeck
- altair
- geopandas
- shapely
- requests

You can install them using:

```bash
pip install streamlit pandas numpy pydeck altair geopandas shapely requests
```
After installing the packages:  
In terminal use command `streamlit run app.py` to launch the dashboard.
